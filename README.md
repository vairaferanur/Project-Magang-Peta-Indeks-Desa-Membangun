# Peta-Indeks-Desa-Membangun-Kabupaten-Purwakarta
Penyusunan peta Indeks Desa Membangun (IDM) sebagai visualisasi status pembangunan desa.

## Deskripsi
Repository ini hasil kegiatan magang yang dilaksanakan di bawah koordinasi Dinas Komunikasi dan Informatika Kabupaten Purwakarta, dengan penugasan di Dinas Pemberdayaan Masyarakat dan Desa Kabupaten Purwakarta. Peta Indeks Desa Membangun (IDM) disusun untuk menampilkan status pembangunan desa berdasarkan nilai IDM. Informasi yang di sajikan pada peta ialah batas administrasi wilayah dan klasifikasi status IDM desa. Peta IDM digunakan sebagai media visualisasi untuk menggambarkan kondisi pembangunan desa secara umum.

## Kegiatan
- Pengolahan dan penyajian data spasial
- Penyusunan peta tematik

## Tools
- QGIS
- ArcGIS
- Microsoft Excel

## Output
Peta Indeks Desa Membangun (IDM) Kabupaten Purwakarta Tahun 2024
